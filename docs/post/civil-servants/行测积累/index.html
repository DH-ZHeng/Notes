成语解析#

  • 哗众取宠：以浮夸的言论迎合群众，骗取群众的信赖和支持。
  • 不容置喙：语气坚定，不容许别人插嘴。
  • 可见一斑：见到事物的一小部分也可能推知事物的整体。
  • 毋庸讳言：不用隐讳，可以直说的内容。
  • 改弦更张：改革制度或变更方法。
  • 奇货可居：比喻拿某种专长或独特的东西作为资本，等待时机，以捞取名利地位，体现了从中获利的行为。
  • 洛阳纸贵：著作有价值，流传广。
  • 洞见肺腑：很清楚看见人的内心，形容襟怀坦白，待人诚恳。
  • 举重若轻：比喻能力强，能够轻松胜任繁重工作或处理困难的问题。
  • 吹毛求疵：故意挑剔别人的缺点。
  • 虎穴得子：诗文贴切中肯，且深的要旨。
  • 缘木求鱼：方向或办法不对头，不可能达成目的。
  • 目无全牛：眼中没有完整的牛，只有牛的筋骨结构，形容人的技艺高超，已经达到非常纯熟的地步。
  • 筑巢引凤：提供良好条件，吸引人才、技术，资金等。
  • 腾笼换鸟：经济发展过程中的一种战略举措，就是把现有的传统制造业从目前的产业基地“转移出去”，再把“先进生产力”转移进来。
  • 小国寡民：政府小，不轻易劳役人民。
  • 大义凛然：勇士受屈时或烈士遇难前的英勇气概，形容英勇无畏。
  • 虚怀若谷：胸怀像山谷那样深而且宽广，形容十分谦虚。
  • 杳如黄鹤：传说中仙人骑着黄鹤飞去，从此不再回来，现比如无影无踪或下落不明。
  • 似水流年：形容时间一去不复返。
  • 羔羊跪乳/乌鸦反哺：感恩父母、奉孝长辈。
  • 惊鸿一瞥：匆匆看了一眼，却给人留下强烈、深刻的印象，常用来形容美女或所仰慕的女子动人心魄的目光。
  • 魂不附体：形容恐惧万分。
  • 标新立异：提出新的主张、见解或创造出新奇的样式，也指为了显示自己，故意显出自己的与总不同或用往常不同的表达方式来吸引人。
  • 特立独行：泛指特殊的、与众不同的，形容人的志向高洁，不同流俗。
  • 犬牙交错：交界线曲折，像狗牙一样参差不齐，也比喻情况糟糕。
  • 不刊之论：刊，消除/磨灭。用来形容文章或言辞的精准得当。
  • 瑕瑜互见：有缺点也有优点。表示客观的评价。
  • 字字珠玑：说话或写文章言简意赅。
  • 栉风沐雨：形容人经常在外面不顾风雨地辛苦奔波。
  • 不以为然：不认为是对的，表示不同意或否定。
  • 不以为意：不放在心上，表示对人或事抱轻视的态度。
  • 别无长物：除一身之外再没有多余的东西，形容贫穷。
  • 一无所长：没有一点专长。
  • 囊萤映雪：形容刻苦攻读。
  • 汗牛充栋：形容藏书极多。
  • 浩如烟海：多指书籍、文献等数量多，极其丰富。
  • 有的放矢：言论、行动目标明确。
  • 明日黄花：比喻已失去新闻价值的报道或已失去应时的事物。
  • 刻舟求剑：比喻办事刻板拘泥，不知根据实际情况处理问题。
  • 方兴未艾：新生事物正在蓬勃发展。
  • 曲意逢迎：想方设法讨好别人。
  • 马放南山：比如天下太平，不再用兵。
  • 厝火积薪：把火放在柴堆下面，比喻潜伏着很大的危险。
  • 叶公好龙：比喻表面上爱好或拥护，实际上害怕或反对，比喻表里不一。
  • 郑人买履：教条主义，做事死板，不会变通。
  • 兔死狐悲：为同盟的死亡或不幸而伤心。
  • 乐天知命：认为自己的一切都由命运支配，暗语现状。
  • 黄发垂髫：指老人和儿童。
  • 窈窕淑女：美丽而有品行的女子。
  • 婷婷玉立：女子身材细长。
  • 长袖善舞：条件好，事情就容易成功。现也比喻有手段的人善于取巧钻研。
  • 挂一漏万：形容列举不周，选了一个，但遗漏很多。
  • 盲人摸象：比喻对事物了解不全面，以偏概全，乱加揣测。
  • 不一而足：形容同类的事物很多，不能一一列举。
  • 添枝加叶：比喻夸大事实或在原有事实上编造一些细节。
  • 火中取栗：受人利用，为别人冒险，徒然吃苦而得不到好处。
  • 张弛有道：比喻工作的紧松和生活的要义要适当调节，有节奏地进行，强调劳逸结合，而非适度。
  • 休戚与共：彼此共同承受幸福与灾祸，形容彼此之间厉害关系密切。
  • 别具匠心：指独特的、巧妙的构思，多指文学、艺术方面创造性的构思。
  • 独具匠心：指具有与众不同的巧妙构思，可形容充满美感的建筑。
  • 别出心裁：形容诗文、美术、建筑等的构思设想别具一格，与众不同。
  • 浑然天成：形容诗文结构严密自然，用词运典毫无斧凿痕迹。亦形容人的才德完美自然。
  • 诘屈聱牙：形容文字拗口难懂、艰涩生僻，体现了晦涩、枯燥。
  • 不忍猝读：形容文章内容悲惨动人。
  • 妙手偶得：技术高超的人，偶然间得到的灵感或作品。
  • 妙笔生花：指笔法高超的人写出动人的文章，强调“技术高超”。
  • 自惭形秽：指因为自己不如别人而感到惭愧。
  • 和光同尘：指不露锋芒，与世无争的处事态度。
  • 防微杜渐：坏事情、坏思想萌芽的时候就加以制止，不让它发展。
  • 投鼠忌器：比喻做事有顾虑，不敢放手干，可形容为了保护隐私而不敢去探索新事物。
  • 噤若寒蝉：受到压制不敢作声。
  • 披沙拣金：比喻从大量的东西中选取精华。
  • 大浪淘沙：比喻在激烈的斗争中经受考验、筛选。
  • 老调重弹：比喻把说过多次的理论、主张重新搬出来。
  • 激浊扬清：清除坏的，发扬好的。
  • 心驰神往：形容思想集中在追求和向往的事情或地方上，一心向往。
  • 煮鹤焚琴：比喻糟蹋美好的事物。
  • 锦衣夜行：享有荣华富贵不在人前显示。
  • 故步自封：守着老一套，不求进步。
  • 水中捞月：比喻去做根本做不到的事情，只能白费力气。
  • 簪笔磬折：古代插笔备礼，曲体作揖，以示恭敬。
  • 耸人听闻：夸大事实或说离奇的话使人感到震惊。
  • 乏善可陈：说不出有什么优点，没什么好称道的。
  • 陈陈相因：比喻沿袭老一套，无创造革新。
  • 阳春白雪：比喻高深的不通俗的文学艺术。
  • 顾影自怜：形容孤单失意的样子，也指自我欣赏。
  • 穿凿附会：指把讲不通的或不相干的道理、事情硬扯在一起进行解释。
  • 博观约取：广博读书而简约审慎地取用。
  • 生生不息：指不断生长、繁殖，多搭配文化、精神等。
  • 削足适履：不合理地迁就凑合，或不顾具体条件地生搬硬套。
  • 大开眼界：开阔事业，增长见识，强调获取新的知识与见闻。
  • 拍案叫绝：拍着桌子叫好，形容非常赞赏，更强调对某人的作品或者话语意见赞赏。
  • 临渴掘井：比喻不早做准备，事到临头才仓促想办法。
  • 运筹帷幄：在后方决定作战方案，也泛指主持大计。
  • 孜孜以求：不知疲倦地追求。
  • 薪火相传：比喻学问技艺一代代地继承下去，也比喻种族、文化等代代相传。
  • 前赴后继：前面的人上去，后面的人就跟上去，形容奋勇前进，连续不断。
  • 莫衷一是：各有各的意见，不能得出一致的结论。
  • 各行其是：按照格子认为对的去做，比喻各搞一套，侧重强调做法。
  • 积非成是：长期所形成的的错误，往往被当做正确的。
  • 浮光掠影：观察不细致，得到的只是事物的一些表面现象，侧重观察的结果是印象不深刻。
  • 望洋兴叹：在伟大事物面前感叹自己的渺小，也比喻做事时因力不胜或没有条件而感到无可奈何。
  • 离弦走板：箭离开弓弦，唱戏不合板眼，言行偏离公认标准。
  • 大势所趋：名词，强调趋势。
  • 势在必行：强调行动，…政策势在必行。
  • 顺势而为：强调顺势。
  • 因势利导：强调积极主动地引导。
  • 立竿见影：收效非常迅速，主体通常是对策、措施、方法，常搭配效果、效应。
  • 一蹴而就：很容易取得成功，主体常为事情、问题，多用于否定句。
  • 弃若敝屣：像扔破鞋一样扔掉，比喻毫不可惜地扔掉或者抛弃。
  • 奉为圭臬：把某些言论或事当成自己的准则。
  • 狗尾续貂：比喻拿不好的东西接到好的东西后面，显得好坏不相称（多指文学作品）。
  • 削足适履：形容够分迁就现成条件生搬硬套。
  • 去芜存菁：除去杂质、保留精华。
  • 讳疾忌医：怕人批评而掩饰自己的缺点和错误。
  • 无所适从：不知听从哪一个好，往往用于前文出现多种观点选择的语境。
  • 以邻为壑：拿邻国当作大水坑，把本国的洪水排泄到那里去，比喻把困难或灾祸推给别人。
  • 独善其身：做不上官，就搞好自身的修养。现在也指只顾自己，缺乏集体精神。

词语辨析#

  • 造就：指培养使成才。铸就：铸成，造就（常用与比喻）。
  • 移植：比如引进别处经验、长处、做法。
  • 笃定：心理踏实，有把握。
  • 盛景：盛大的景象，侧重盛大。胜景：指优美的风景。
  • 晚造：晚季作物。
  • 南冠：囚犯。
  • 浮屠：佛塔。
  • 袍泽：军中同事。
  • 滥觞：本谓江河发源之处水极浅小，仅能浮起酒杯，后比喻事物的起源和发端。
  • 兀立、矗立、屹立：直立之意。伫立：长时间站立，没有动作，指人的站立。
  • 嬗变：蜕变、演变，指本质上的变化。eg:中国传统文化在道光、咸丰、同治年间进入了嬗变的关键期。
  • 掩映：指彼此遮掩而相互衬托，时隐时现。
  • 福祉：幸福、利益、福利。
  • 延绵：指形态上。延绵起伏的高山。
  • 延续：时间上的连续、不中断。
  • 赓续：指继续，多搭配“精神”、“传统”等。
  • 演进：指演变发展。可搭配历史。
  • 演变：变化发展。可搭配历史。
  • 演绎：铺陈、表现，比喻从前提必然地得出结论的推理。
  • 基础：指事物发展的根本，可体现其重要作用。实质：指某一对象或事物本身所必然固有的性质，或事物的内在含义，强调事物的本质特点，与重要性无关。
