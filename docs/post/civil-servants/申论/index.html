分析答题点

  • 分数
    20分：可能4或者5个要点
    25分：可能5个要点
  • 字数
    平均一个词5个字
    总字数除以5，乘以2/3一个词平均是5，乘以2/3是为了去掉空格
    150字：20词
    200字：25词
    300字：35词
    再平均分配到要点上

分析材料

  • 读材料是要整体把握材料分了哪几个部分，根据题目的字数确定每个答题点的词数

  • 材料中有人讲话的地方非常重要，一般有答题点，讲话分为两类

    讲话类型 做题策略
    直接指出问题/对策 直接抄
    讲故事 需要概括归纳

  • 不知道写分类词的时候，从一头一尾挑分类词的正确率最大

单一题

  • 特点

    1. 作答要素明确：问题、对策、意义、变化、特点
    2. 作答要素相对较少

  • 答题模板

    1. 答案分条列点("1.2.3.")
    2. 每条要点上话题一致，有总括性信息

1.加强基层党建。壮大队伍，选派优秀人才；将基层党建组织下沉。
2.改善人居环境。推进农村环境综合整治和生态文明建设，提升环境。
3.保护历史文化。修缮和活化文化遗存。

对策是一个动作

  • 出题方式

    1. 概括做法：做法已经做了，做了概括出来，本质上就是抄。
    2. 启示/经验：
    Example
    ①小明这个做法做得特别好，谈一谈小明的做法对别人有什么样的启示/可供借鉴的经验
    ②小明的做法已经做了，也是抄，但是抄的过程中注意普适性

    比如小明好公务员，同学考的是教资。小明做的是申论题，同学做的是教资题，需要找到二者之间的共同性/普适性

    1. 根据问题提对策：这个对策没有发生，不能抄，此时是推对策

  • 概括举措

    1. S市已经做过了，可以直接抄
    2. 难点：字数很少，但材料特别多
    3. 取舍的重点是逻辑：识别出总结性强的词
    4. 出现标志性动词时，往往得分点
    常见动词
    建立、健全、统筹、创新、规范、整顿、打击、扶持、规划、设置等

    对策是动宾短语

  • 经验启示类

    1. 答题技巧：从其他主题(其他地区、国家、企业等)的成功经验中借鉴。通常而言，借鉴而来的经验要具有普适性。

    L县做得很好，对M县的启示，注意如果L县卖苹果，M县卖梨，需要将苹果相关词去掉，涉及到迁移性问题

    并不是所有经验启示类都需要迁移、普适，真题中只有两种情况需要普适

    情况 需要普适原因
    M农场的案例为新时代青年创业提供了哪些启示？ 小→大（个例到全局）
    芬兰政府鼓励创新的做法对我国政府在科技创新中应该发挥什么作用提供哪些有益的启示？ 不同国别/领域
    • 不需要普适的情况：M县的扶贫工作对G市其他县...（同等级别）

  • 提出对策类

    1. 提出对策类的题目是针对没有发生的，需要自己根据问题推出对策
    2. 答题技巧
      通过问题反推
      反推的对策优先级1：常用对策动词+问题名词【没有非机动车道--增设非机动车道】
      反推的对策优先级2：推一步,根据背后原因推对策【不认同原因--观念出了问题,观念需要宣传】

    反推第二种情况是由于问题属于表象问题，应该从核心解决问题
    饿了→解决方法不应该是“不饿“, 应该推一步，”去吃饭“

积累

  • 规划：先收集意见、走访调研，然后形成方案/办法/模式/思路/治理理念/政绩观。
  • 监督：1.依靠法制/制度监督。如整治互联网乱象，首先要立法，填补法律空白，其次需要提高违法成本。2.依靠舆论监督。民众举报/关注、媒体曝光、信息公开透明。
  • 劝/哄：普及知识、转变观念、培训技能、沟通对话、宣传引导。
  • 监督处罚：监督是依靠法律约束，监督是直接进行打击处罚。为了有效监督，统一协调、全责分明、联合执法、源头监管、全过程监管。
  • 投入：提供资金、技术、人，优化服务、提升服务态度。

综合题

逻辑优先

词句解释题

  • 题型判断
    谈...的含义； 对...进行解释； 谈对...的理解
  • 常规思路
graph LR A[词句解释]--->B1(释义) B1--->C11(题干:字面含义自己要理解) B1--->C12(材料:这句话前后多看看) A--->B2(要素) B2--->C21(根据题干预判问题原因等) B2--->C22(根据材料找出要点或者梳理逻辑) A--->B3(对策) B3--->C31(字数够,有就抄,没有就反推) B3--->C32(字数不够,不用强求)

细节注意

  • 单一题读材料先整体后部分，是为了看可以分几条
  • 综合题读材料需要优先找出词汇/句子是什么意思，优先看划线句子/词汇所在位置
  • 综合题里面开头讲故事的可以先不总结，后文可能有总结的内容
  • 专家说话是重点

作答思路

  • 下定义（2行)
  • 具体展开（重点）
  • 总结性对策（2行)

  • 判断中间部分作答内容的方式：
    由题干给出句子/词汇的感情色彩来决定，好的内容需要作答好的做法\意义

答题技巧

  • 问题出现了截然不同的表述，要分开作答

"想着懒，而不是想得懒"
这句话指的是...
想着懒还带来好处：...
想得懒则会带来诸多问题:...
这就要要求我们勤于思考...

  • 中间内容包括两部分：题干句子有好有坏，想着懒可以带来好处，懒得想会带来问题/危害。一句话中出现截然相反/差异性较大的两种意思，中间展开部分需要分开表述。

观点现象分析题

  • 题型判断
    对...观点/现象的见解/看法/评价/评析/认识
    观点明确、分析透彻、论证充分、有理有据、论据充实等

常规思路

  • 表态+理由+对策
  • 表态：要依据题干和资料，而非个人主观判断
  • 理由：围绕主题，写出资料中能够支撑态度的各种要素
  • 对策：资料中有则摘抄；没有的话，字数允许则补充
高分技巧
赞同的 正确的、科学的、合理的、全面的等
反对的 不科学的、不合理的、不支持的、错误的、偏颇的、偏激的、绝对的、片面的

公文题

格式要求

  • 公文题一般会要求写“一份关于...的知道意见” 或“一篇在...上的讲话稿”
  • 文种体现：意见、讲话稿
  • 只要公文就要有格式，除非题目要求是提纲、纲要内容要点或不注重格式时才不需要有格式

阅卷要求

  • 格式分：标题、称谓（主送机关）、落款

  • 内容分：问题、危害、意义、对策等

  • 语言分：通俗易懂、表达生动、有感染力、有号召力、态度诚恳、建议口吻等

  • 标题

    • 一般写法：发文机关+“关于”+是由+“的”+文种。
      eg：国务院关于坚决稳定市场物价的通知
    • 书写格式
      标题要居中书写（不带书名号）
    • 公文题的标题要如何写，要分场合，可以通过协作对象来判别场合、
    对象 公文标题
    政府内部、领导、机关单位（正式场合） 一般写法
    市民、老百姓、学生（相对灵活） 自拟的、活泼的标题

  • 称谓或主送机关

    • 称谓——对象为任务
      eg：尊敬的与会嘉宾、广大家长朋友们等
    • 主送机关（对象为单位）
    • eg：各社区、各街道办等
    • 定格书写

  • 落款

    • 单位（个人）
      eg：某企业工会（时间上一行）
    • 时间
      eg：2022年XX月XX日或者XXXX年XX月XX日
    • 落款位于正文右下方，日期有空四个格子，发文者以日期为准上方居中
  • 公文有无格式对应表
    题目 标题 称谓 落款
    报告、通告、意见、通知等 有 有 有
    倡议书、公开信、宣传稿 有 有 有
    经验发言稿、事件处理发言稿 有 有 根据格子确定
    时评、网评、新闻稿等 有 无 无
    调查问卷、活动方案等 有 无 无
    编制按、内容要点 无 无 无
    明确要求不懈公文格式的命题方式 无 无 无

具体类别

方案类

  • 识别
    一般侧重对策、建议、具体做法（看到问题，去提出工作的建议）

  • 典型类型
    指导意见、建议、活动方案等

  • 公文题开头要先有一个帽子
    交代背景，方案类帽子写一些问题+具体方案如下/建议如下
    帽子字数占比1/3

  • 细节注意
    结尾可以加上一句：以上建议请领导参考
